- Tap a bar — it highlights orange, "Selected" label updates
- Tap "+50" button — selected bar grows upward, value updates
- Tap "-50" button — selected bar shrinks or goes negative (below zero line)

**Overall:**
- Dark theme consistent with rest of app
- All text is readable, no overlapping elements
- Touch targets feel appropriately sized (no tiny buttons)
