GREEN phase: Add four methods to AuthManager in src/dmccodegui/auth/auth_manager.py:

1. `create_user(self, name: str, pin: str, role: str) -> Optional[str]`:
   - Validates: name non-empty, name not already in users, PIN is 4-6 digits, PIN not already used by another user
   - Adds to self._data["users"][name] = {"pin": pin, "role": role}
   - Calls self._save()
   - Returns None on success, error string on failure

2. `delete_user(self, name: str) -> Optional[str]`:
   - Validates: name exists, if role is admin then count admins > 1
   - Deletes from self._data["users"]
   - Calls self._save()
   - Returns None on success, error string on failure

3. `update_user(self, name: str, new_name: Optional[str] = None, new_pin: Optional[str] = None, new_role: Optional[str] = None, current_user: Optional[str] = None) -> Optional[str]`:
   - Validates role change: blocks self-demotion (name == current_user), blocks last-admin demotion
   - Validates PIN: 4-6 digits, not used by another user (exclude self from duplicate check)
   - Validates name change: non-empty, not duplicate
   - Performs name rename: copies dict entry, deletes old key, updates last_user if it matched old name
   - All validation runs BEFORE any mutation
   - Calls self._save()
   - Returns None on success, error string on failure

4. `get_all_users(self) -> list[dict]`:
   - Returns list of {"name": str, "role": str, "pin_masked": str} where pin_masked is bullet character repeated per PIN digit count
