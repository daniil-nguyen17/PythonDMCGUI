**DeltaCBarChart class** (add before RunScreen class in run.py):

A `Widget` subclass that draws vertical bars using Kivy canvas instructions. Properties:
- `offsets = ListProperty([])` — one float per section, accumulated offset from zero
- `selected_index = NumericProperty(-1)` — which bar is selected (-1 = none)
- `max_offset = NumericProperty(500)` — max offset for scaling bar height

Methods:
- `on_offsets`, `on_selected_index`, `on_size`, `on_pos` — all call `self._draw()`
- `_draw()`: Clear canvas. If no offsets, return. Calculate bar width = `self.width / len(self.offsets)`. Calculate mid_y = `self.y + self.height / 2`. Draw a thin horizontal zero-line across the middle (Color grey, Rectangle height 1dp). For each offset: calculate bar_h proportional to `abs(offset) / max_offset * (self.height / 2)`, clamp to minimum 2px. Use highlight color (orange: `1.0, 0.65, 0.0, 1`) if selected, default color (blue: `0.235, 0.510, 0.960, 1`) otherwise. Draw Rectangle above mid_y for positive, below for mid_y for negative. Draw offset value text above/below each bar using `Label` would be complex in canvas — instead, the selected bar's value is shown in a separate Label widget in KV.
- `on_touch_down(touch)`: If touch collides with this widget, calculate which bar index was tapped from `(touch.x - self.x) / bar_width`. Set `self.selected_index` to that index. Return True to consume touch.

**RunScreen adjustment methods** (add to RunScreen class):

Properties to add:
- `section_count = NumericProperty(5)` — default 5 sections
- `delta_c_offsets = ListProperty([0.0] * 5)` — one offset per section
- `selected_section_value = StringProperty("0")` — display value of selected bar

Methods:
- `on_section_count_change(value)`: Clamp value to 1-10. Resize `delta_c_offsets` list to new count (zero-fill new entries, truncate excess). The DeltaCBarChart.offsets property is bound to this in KV.
- `_on_chart_selection_changed(chart_widget, selected_index)`: Observer method bound to `delta_c_chart.selected_index` in `on_kv_post`. When `selected_index >= 0 and selected_index < len(self.delta_c_offsets)`, set `self.selected_section_value = str(int(self.delta_c_offsets[selected_index]))`. When `selected_index < 0`, set `self.selected_section_value = "0"`. Bind in `on_kv_post`: `self.ids.delta_c_chart.bind(selected_index=self._on_chart_selection_changed)`.
- `on_adjust_up()`: If a bar is selected (get `idx = self.ids.delta_c_chart.selected_index`, check `idx >= 0`), add DELTA_C_STEP to `delta_c_offsets[idx]`. Update the list property (must reassign to trigger binding: `self.delta_c_offsets = list(self.delta_c_offsets)`). Update `self.selected_section_value = str(int(self.delta_c_offsets[idx]))`.
- `on_adjust_down()`: Same but subtract DELTA_C_STEP.
- `on_apply_delta_c()`: Convert section offsets back to the 100-element deltaC array. Each section covers `DELTA_C_ARRAY_SIZE / section_count` indices (integer division). For section i, indices `WRITABLE_START + i * chunk_size` through `WRITABLE_START + (i+1) * chunk_size - 1` get that section's offset value. Only write indices within `[WRITABLE_START, WRITABLE_END]`. Submit via `jobs.submit`: call `self.controller.download_array("deltaC", DELTA_C_WRITABLE_START, values_list)`.
- `_offsets_to_delta_c()`: Helper that expands section offsets into the full array slice. Returns a list of floats of length `WRITABLE_END - WRITABLE_START + 1`.

Do NOT add a Reset button (TBD per CONTEXT.md — depends on knowing which indices are writable vs protected).
