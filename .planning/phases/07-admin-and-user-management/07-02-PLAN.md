Create src/dmccodegui/screens/users.py with try/except ImportError guard (same pattern as profiles.py):

**UsersScreen(Screen):**
- Properties: auth_manager = ObjectProperty(None, allownone=True), state = ObjectProperty(None, allownone=True)
- on_pre_enter: call _rebuild_cards() to populate user list from auth_manager.get_all_users()
- _rebuild_cards(): clear cards_container, iterate get_all_users(), create a card per user with: name label, role badge (colored label), masked PIN dots, Edit button, Delete button
- on_add_press(): instantiate UserEditOverlay, call open_for_add(auth_manager, current_user=state.current_user, on_saved=_rebuild_cards)
- on_edit_press(name): instantiate UserEditOverlay, call open_for_edit(auth_manager, name, current_user=state.current_user, on_saved=_rebuild_cards)
- on_delete_press(name): show confirmation Popup with "Delete user '{name}'?" and Cancel/Delete buttons. On confirm: call auth_manager.delete_user(name). If error, show error popup. If success and state.current_user == name, call state.set_auth("", "operator") to force logout. Then _rebuild_cards().

**UserEditOverlay(ModalView):**
- Properties: error_msg = StringProperty(""), target_name = StringProperty(""), auth_manager = ObjectProperty, on_saved = ObjectProperty, current_user = StringProperty("")
- auto_dismiss=False, size_hint=(None, None), width="500dp", height="480dp"
- open_for_add(): set target_name="", clear fields, set role selector default to "operator", open()
- open_for_edit(name): set target_name=name, pre-fill name field, leave PIN field empty (Admin types new PIN or leaves blank to keep existing), pre-select current role, disable role selector buttons if name == current_user (self-demotion guard per user decision). open()
- _select_role(role): update button backgrounds -- selected gets accent color, others get bg_row. Store self._selected_role.
- press_save(): read name_input.text.strip(), pin_input.text.strip(), _selected_role. If target_name == "" (add mode): call auth_manager.create_user(name, pin, role). If target_name != "" (edit mode): call auth_manager.update_user(target_name, new_name= if changed, new_pin= if non-empty, new_role=, current_user=current_user). On error: set error_msg, clear after 3s. On success: dismiss(), call on_saved().

- setup: orange (1.0, 0.65, 0.0, 0.9)
- admin: cyan (0.0, 0.8, 0.85, 0.9)

Card styling: use dark background (theme.bg_card or slightly lighter than bg_dark), rounded corners if CardFrame available, left-edge role color stripe. Name in large font (20sp), role badge pill, masked PIN in muted color, Edit/Delete buttons on right side (44dp+ touch targets).
