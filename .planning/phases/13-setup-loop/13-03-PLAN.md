**1. varcalc integration in `apply_to_controller`:**
Modify the `_job()` inner function. After writing all dirty params (the existing `for var_name, text in dirty_snapshot` loop), insert:
```python
# Fire varcalc trigger
try:
    ctrl.cmd(f"{HMI_CALC}={HMI_TRIGGER_FIRE}")
except Exception:
    pass

# Wait for #VARCALC to complete on controller
import time
time.sleep(0.5)
```
Then the existing readback loop (`for p in param_defs_snapshot`) stays in place. Then BV. The sequence becomes: write params -> fire hmiCalc=0 -> sleep 0.5 -> read back all -> BV.

**2. Smart enter (`on_pre_enter`):**
Replace the unconditional `controller.cmd(f"{HMI_SETP}={HMI_TRIGGER_FIRE}")` with a state guard:
```python
already_in_setup = (
    self.state is not None and self.state.dmc_state == STATE_SETUP
)
if self.controller is not None and self.controller.is_connected():
    if not already_in_setup:
        try:
            self.controller.cmd(f"{HMI_SETP}={HMI_TRIGGER_FIRE}")
        except Exception:
            pass
```
The `read_from_controller()` call stays unconditional (always refresh values on entry).

**3. Smart exit (`on_leave`):**
Replace the `hmiSetp=HMI_TRIGGER_DEFAULT` reset with smart exit:
```python
_SETUP_SCREENS = frozenset({"axes_setup", "parameters"})

def on_leave(self, *args):
    next_screen = ""
    if self.manager:
        next_screen = self.manager.current
    if next_screen not in _SETUP_SCREENS:
        if self.controller is not None and self.controller.is_connected():
            try:
                self.controller.cmd(f"{HMI_EXIT_SETUP}={HMI_TRIGGER_FIRE}")
            except Exception:
                pass
```
Define `_SETUP_SCREENS` as module-level frozenset (same as axes_setup.py pattern).

IMPORTANT: The `is_connected()` check must be used consistently. The existing parameters.py code uses `self.controller is not None` without `is_connected()` in on_pre_enter — update to add `is_connected()` for the hmiSetp fire, but keep `read_from_controller()` as-is (it has its own connection guard internally).
