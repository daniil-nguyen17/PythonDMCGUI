From app_state.py — MachineState subscription:
```python
def subscribe(self, fn: ChangeListener) -> Callable[[], None]:
    """Subscribe to state changes. Returns unsubscribe callable."""
    self._listeners.append(fn)
    def unsubscribe() -> None:
        try:
            self._listeners.remove(fn)
        except ValueError:
            pass
    return unsubscribe
```
