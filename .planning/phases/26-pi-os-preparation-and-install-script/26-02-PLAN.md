    **STEP 3: Create deploy/pi/install.sh (GREEN)**

    The install script must follow these locked decisions exactly:
    - **Shebang and strict mode:** `#!/usr/bin/env bash` + `set -euo pipefail`
    - **Self-elevating:** Check $EUID, print "Please run with sudo" and exit if not root
    - **Architecture detection:** Detect aarch64 via `uname -m`. Print informational message that Kivy will compile from source (takes 20-40 minutes). Do NOT abort — user decision targets 64-bit.
    - **Constants:** INSTALL_DIR="/opt/binh-an-hmi", VENV_DIR="$INSTALL_DIR/venv", LOG_FILE="/var/log/binh-an-hmi-install.log", SCRIPT_DIR derived from BASH_SOURCE
    - **log() helper:** `echo "==> $*" | tee -a "$LOG_FILE"` for step banners
    - **Step ordering (per locked decision: X11 -> apt -> gclib -> venv -> pip):**

    Step 1 - Force X11:
    ```bash
    log "Forcing X11 session (Kivy requires X11) ..."
    raspi-config nonint do_wayland W1
    ```

    Step 2 - Disable screen blanking:
    ```bash
    log "Disabling screen blanking (always-on HMI) ..."
    raspi-config nonint do_blanking 1
    ```

    Step 3 - Enable SSH:
    ```bash
    log "Ensuring SSH is enabled ..."
    systemctl enable ssh
    systemctl start ssh
    ```

    Step 4 - apt dependencies (CRITICAL for aarch64):
    ```bash
    log "Installing apt dependencies ..."
    apt-get update -qq 2>&1 | tee -a "$LOG_FILE"
    apt-get install -y \
        python3 python3-pip python3-venv \
        build-essential cmake pkg-config \
        libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev \
        libgl1-mesa-dev libgles2-mesa-dev libegl1-mesa-dev \
        libmtdev-dev libxrender-dev \
        gstreamer1.0-plugins-base gstreamer1.0-plugins-good \
        libgstreamer1.0-dev \
        2>&1 | tee -a "$LOG_FILE"
    ```
    Note: On aarch64, Kivy compiles from source. It needs SDL2 dev headers, OpenGL dev headers, and GStreamer dev libs. build-essential provides gcc/g++, cmake is needed by Kivy's build system, pkg-config resolves library paths. These deps are harmless on armhf (pip just uses the pre-built wheel and ignores them).

    Step 5 - Galil gclib:
    ```bash
    log "Installing Galil gclib system library ..."
    if ! dpkg -l gclib 2>/dev/null | grep -q "^ii"; then
        dpkg -i "$SCRIPT_DIR/vendor/galil-release_1_all.deb" 2>&1 | tee -a "$LOG_FILE"
        apt-get update -qq 2>&1 | tee -a "$LOG_FILE"
        apt-get install -y gclib gcapsd 2>&1 | tee -a "$LOG_FILE"
    else
        log "gclib already installed — skipping"
    fi
    ```

    Step 6 - Copy app to /opt:
    ```bash
    log "Installing app to $INSTALL_DIR ..."
    mkdir -p "$INSTALL_DIR"
    rsync -a --delete "$SCRIPT_DIR/../../" "$INSTALL_DIR/" \
        --exclude='.git' --exclude='deploy/' --exclude='dist/' --exclude='build/' \
        --exclude='.planning/' --exclude='tests/' --exclude='__pycache__' \
        2>&1 | tee -a "$LOG_FILE"
    ```

    Step 7 - Create venv (idempotent):
    ```bash
    if [[ ! -d "$VENV_DIR" ]]; then
        log "Creating Python venv at $VENV_DIR ..."
        python3 -m venv "$VENV_DIR" 2>&1 | tee -a "$LOG_FILE"
    else
        log "Venv already exists — skipping creation"
    fi
    ```

    Step 8 - pip install:
    ```bash
    log "Installing Python dependencies (this may take 20-40 minutes on 64-bit Pi OS) ..."
    "$VENV_DIR/bin/pip" install --upgrade pip 2>&1 | tee -a "$LOG_FILE"
    "$VENV_DIR/bin/pip" install -r "$INSTALL_DIR/deploy/pi/requirements-pi.txt" \
        2>&1 | tee -a "$LOG_FILE"
    ```
    Note: On aarch64 without PiWheels, Kivy will compile from source. The apt deps from Step 4 provide the required headers. The "20-40 minutes" message sets expectations.

    Step 9 - Desktop shortcuts:
    ```bash
    log "Creating desktop shortcuts ..."
    cp "$SCRIPT_DIR/binh-an-hmi.png" "$INSTALL_DIR/binh-an-hmi.png"
    cp "$SCRIPT_DIR/binh-an-hmi.desktop" /usr/share/applications/binh-an-hmi.desktop
    REAL_USER="${SUDO_USER:-pi}"
    REAL_HOME=$(getent passwd "$REAL_USER" | cut -d: -f6)
    mkdir -p "$REAL_HOME/Desktop"
    cp "$SCRIPT_DIR/binh-an-hmi.desktop" "$REAL_HOME/Desktop/binh-an-hmi.desktop"
    chmod +x "$REAL_HOME/Desktop/binh-an-hmi.desktop"
    gio set "$REAL_HOME/Desktop/binh-an-hmi.desktop" metadata::trusted true 2>/dev/null || true
    chown "$REAL_USER:$REAL_USER" "$REAL_HOME/Desktop/binh-an-hmi.desktop"
    ```

    Step 10 - Summary and reboot:
    ```bash
    log ""
    log "============================================"
    log "  Installation complete!"
    log "  App: $INSTALL_DIR"
    log "  Venv: $VENV_DIR"
    log "  Data: ~/.binh-an-hmi/ (created on first launch)"
    log "  Log: $LOG_FILE"
    log "============================================"
    log ""
    log "Rebooting in 10 seconds (Ctrl+C to cancel) ..."
    for i in $(seq 10 -1 1); do printf "\r  %ds " "$i"; sleep 1; done
    echo ""
    reboot
    ```
