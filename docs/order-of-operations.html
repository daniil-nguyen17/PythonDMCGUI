3. GRINDING CYCLE

Running the actual grind — operator monitors from RUN screen

1
Press START GRIND
HMI fires hmiGrnd=0. DMC enters #GRIND. State = GRINDING (2). HMI disables Start button, shows STOP button.
User ActionhmiGrnd=0DMC #GRIND
|
2
DMC executes grind sequence
Phase 1: Move to start position (#GOSTR), set D angle + compensate C
Phase 2: Vector contour — feed deltaA/B/C/D segments via Linear Interpolation (LI)
Phase 3: A overrun past heel, park to rest (#GOREST)
DMC #GRINDAutomatic
|
3
HMI monitors live
Live A/B position plot updates at 5 Hz. Knife count increments. State bar shows GRINDING (green). Operator watches progress.
RUN Screen10 Hz polling
|
4
Grind completes
DMC increments ctSesKni and ctStnKni, parks to rest, returns to #MAIN. State = IDLE (1). HMI re-enables Start button. Load next knife and repeat.
DMC #DONEhmiState=1

5. E-STOP RECOVERY

How to restart after an emergency stop

1
E-STOP active
Status bar shows E-STOP (red). All axes stopped. DMC program halted. RECOVER button appears on status bar.
AutomaticStatus Bar
|
2
Press RECOVER
HMI sends SH ABCD (servo on) + XQ #AUTO (restart program from top). This is the only authorized XQ call. DMC re-initializes: config, params, home, then enters #MAIN.
User ActionRECOVER ButtonXQ #AUTO
|
3
System restored
DMC homes all axes, enters IDLE. Status bar shows IDLE. Machine ready for normal operation. Re-teach start/rest positions if needed.
Automatic#AUTO → #HOME → #MAIN

HMI Trigger Variable Reference

All one-shot variables — HMI writes 0 to fire, DMC resets to 1 immediately

Variable HMI Screen DMC Subroutine What It Does
hmiGrnd RUN #GRIND Start grinding cycle
hmiSetp Axes Setup / Parameters #SETUP (#SULOOP) Enter setup mode
hmiMore RUN #MOREGRI More stone compensation (+0.001mm)
hmiLess RUN #LESSGRI Less stone compensation (-0.001mm)
hmiNewS RUN #NEWSESS New stone session (home + reset counters)
hmiHome Axes Setup #HOME Home all 4 axes
hmiJog (Physical panel only) #WheelJg Enter handwheel jog mode
hmiCalc Parameters #VARCALC Recalculate knife variables
hmiGoRs Axes Setup #GOREST Move all axes to rest position
hmiGoSt Axes Setup #GOSTR Move all axes to start position
hmiExSt Any (on leave setup) Exits #SULOOP Exit setup, return to MAIN (IDLE)
ONE-SHOT RULE: The HMI NEVER calls XQ #subroutine directly. It always writes variable=0 to fire a trigger. The DMC resets the variable to 1 as its very first action inside each triggered block. The only exception is XQ #AUTO during E-STOP recovery.
HMI Screen Navigation

How the touchscreen screens connect to DMC modes

HMI Screens
  • Login — PIN entry, machine type selection
  • Setup — Controller IP, connect/disconnect
  • RUN — Start/Stop grind, stone comp, delta-C, live plot
  • Axes Setup — Jog axes, teach rest/start points, quick actions
  • Parameters — Edit knife thickness, feed rates, calibration
  • Profiles — Import/export CSV knife profiles
  • Status Bar — Always visible: state, connection, E-STOP button
DMC Modes
  • #MAIN / #WtAtRt — Idle wait loop, polls triggers
  • #GRIND — Full grind cycle (go-start, contour, go-rest)
  • #SETUP / #SULOOP — Setup menu, polls setup triggers
  • #HOME — Home all 4 axes sequence
  • #GOREST / #GOSTR — Safe multi-axis positioning
  • #VARCALC — Recalculate derived values from params
  • #MOREGRI / #LESSGRI — Stone compensation adjustments
SETUP SCREEN SIBLINGS: Navigating between Axes Setup and Parameters does NOT exit setup mode (no hmiExSt=0 sent). Setup mode only exits when navigating to a non-setup screen (RUN, Profiles, etc.).
